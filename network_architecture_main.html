Architecture Overview

This detailed low-level design document outlines the network architecture for a three-datacenter deployment with Distributed Control and Centralized Management. The architecture consists of two active-active datacenters (North and South) with direct internet connectivity and a tertiary datacenter serving as a control hub without direct internet access.

Key Architecture Principles

  • Distributed Control: Each datacenter maintains its own control plane for local operations
  • Centralized Management: Unified management and orchestration from the Tertiary Control Hub
  • Autonomous Operation: Each datacenter can operate independently if isolated
  • Consistent Policy: Standardized policy model across all three datacenters
  • Defense in Depth: Multiple security layers with clear zone separation
  • Resilient Connectivity: Redundant paths between all datacenters

North Datacenter Overview

The North Datacenter serves as one of the two active sites in the active-active DC-DR configuration. It provides direct internet connectivity and hosts production workloads.

Key Components:

  • Internet Edge Infrastructure
  • DMZ Zone
  • ACI Fabric
  • Compute and Storage Resources
  • Local Management Systems

South Datacenter Overview

The South Datacenter serves as the second active site in the active-active DC-DR configuration. Like the North Datacenter, it provides direct internet connectivity and hosts production workloads.

Key Components:

  • Internet Edge Infrastructure
  • DMZ Zone
  • ACI Fabric
  • Compute and Storage Resources
  • Local Management Systems

Tertiary Control Hub Overview

The Tertiary Control Hub serves as the centralized management and orchestration center for the entire infrastructure. It does not have direct internet connectivity and focuses on management, security, and control functions.

Key Components:

  • ACI Multi-Site Orchestrator
  • Network Management Systems
  • Security Operations Center
  • Automation and Orchestration Platforms
  • Backup and Recovery Infrastructure
